Splinter Examples
==================


Splinter is an open source tool for testing web applications using Python.
It lets you automate browser actions, such as visiting URLs and interacting with their items.

If you don't have it yet, go see the `official documentation <http://splinter.readthedocs.io/en/latest/install.html>`_ to get it installed and get started.
